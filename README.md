## Gitter Bot

### Installation

```
   $ npm install 
```

### Configuration

If you want to change **token** you should open 'config.json' file and replace test-token in it. 

Gitter room for testing - **biosis21/uwc-bot-testing**.

### Running

```
   $ node index.js --room=<your_room>
```
